Código Compartilhado

O que é um acordo de código compartilhado?

É um acordo entre duas companhias aéreas através do qual uma companhia pode comercializar voos de outra utilizando o seu próprio código, com o intuito de oferecer mais opções de destinos.

Como reconhecer um vôo com código compartilhado

Quando estiver fazendo cotação de itinerários no site da copa.com, os voos com código compartilhado estarão assinalados com um número de referência ao lado do número de voo. Este número se refere a uma nota no rodapé da página que indica qual é a companhia aérea operadora.

Neste exemplo, o vôo CM5628 possui um indicador para identificar a United Airlines como a companhia aérea operadora na rota LAX-HNL. Como se trata de um código compartilhado, o número de vôo começa com as letras ‘CM’, que representam o código comercializador da Copa Airlines. No seu itinerário eletrônico poderá também identificar a companhia aérea operadora.

Atualmente, a Copa Airlines mantém acordos de código compartilhado com companhias aéreas na Ásia, América e Europa.

Quando efetuar uma reserva em um vôo com código compartilhado, receberá informações oportunas sobre qual será a companhia aérea operadora. As Condições de Transporte podem variar entre as companhias aéreas, pelo que se recomenda revisá-las na página da Internet da operadora.
